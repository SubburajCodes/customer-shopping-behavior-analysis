# Customer Shopping Behavior Analysis

## 📌 Project Overview

This project analyzes customer shopping behavior using transactional data from 3,900+ customer purchases. The objective is to identify spending patterns, customer segments, product preferences, and subscription trends that can support data-driven business decisions.

The project follows a complete Data Analytics workflow including data cleaning, SQL-based business analysis, and interactive dashboard development.

---

## 🎯 Business Objective

The primary goal of this project is to answer key business questions such as:

- Which customer groups generate the highest revenue?
- How do subscribers differ from non-subscribers?
- Which products perform best across categories?
- How do discounts influence customer purchasing behavior?
- Which age groups contribute the most revenue?

## 🔧 Data Cleaning & Preprocessing

The dataset was cleaned and prepared using Python:

- Handled missing values in Review Ratings
- Standardized column names for consistency
- Created Age Group categories
- Removed redundant columns
- Performed data validation checks
- Loaded cleaned data into PostgreSQL for analysis
